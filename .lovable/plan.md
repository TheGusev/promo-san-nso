
# План: Удаление информации о времени работы и адресе из футера

## Что нужно удалить

На скриншоте выделены две строки в футере:
1. **Работаем круглосуточно** (с иконкой часов)
2. **г. Новосибирск, ул. Державина, 28** (с иконкой геопина)

## Где находится этот код

Файл: `src/components/layout/SiteFooter.tsx`, строки 38-47

```tsx
<div className="space-y-2 text-sm">
  <div className="flex items-center gap-2">
    <Clock className="h-4 w-4 text-muted-foreground" />
    <span>{SITE_CONFIG.workingHoursText}</span>
  </div>
  <div className="flex items-center gap-2">

  </div>

```

## Какие страницы затронуты

`SiteFooter` используется через `MainLayout`, который применяется на всех внутренних страницах:
- Страницы услуг (`/usluga/...`)
- Страницы вредителей (`/vreditel/...`)  
- Страницы объектов (`/obekt/...`)
- Страницы районов (`/rayon/...`)
- Блог (`/blog/...`)
- FAQ (`/faq`)
- СанПиН (`/sanpin`)
- И другие

Главная страница использует отдельный `Footer.tsx`, там этой информации нет.
