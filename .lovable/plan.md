

# Калькулятор: упростить до 2 списков с автопереходом

## Что меняем в `src/components/SimpleCalculator.tsx`

### 1. Убрать кнопки «Далее»
- Шаг 1: после выбора в селекте вредителя — **автоматически** переключаемся на шаг 2 (через `setTimeout(…, 150)` чтобы успела проиграться анимация закрытия селекта). Если у вредителя только одно место (кроты, осы и т.п.) — сразу шаг 3.
- Шаг 2: после выбора места — **автоматически** на шаг 3.
- Кнопки «Далее» удалить полностью с обоих шагов.

### 2. Кнопка «Назад» — аккуратная, ненавязчивая
Остаётся одна тонкая ссылка-кнопка слева сверху карточки (`ghost` вариант, маленький размер, иконка стрелки + «Назад»). Уже есть в текущей реализации — оставляем как есть, только убедимся, что выглядит лёгкой (не кнопка-блок).

### 3. Селекты раскрываются ВНИЗ, а не вверх
Сейчас Radix Select по умолчанию использует `avoidCollisions=true` — если внизу мало места, он открывается вверх. Принудительно задаём:
```tsx
<SelectContent position="popper" side="bottom" sideOffset={8} avoidCollisions={false}>
```
для обоих селектов. Так список всегда «выпадает» вниз мягко под триггером.

### 4. Шаг 3 — добавить дисклеймер под ценой
Под крупной ценой добавить мелкую строку:
```
Итоговая стоимость определяется мастером после осмотра на месте
```
Класс: `text-xs text-muted-foreground mt-2`. Показываем всегда (не только для «от N»).

### 5. Облегчить визуал
- Убрать «Шаг X / 3» текст и progress-bar — заменить на 3 маленькие точки-индикатора (•••), активная точка — primary, остальные — muted. Минимально и аккуратно.
- Уменьшить padding карточки на десктопе (`p-6` вместо `p-8`).
- Заголовок «Кто вас беспокоит?» / «Где провести обработку?» сделать поменьше (`text-base md:text-lg`, `font-medium` вместо `font-semibold`).
- Убрать подзаголовок «3 простых шага — узнайте цену и оставьте заявку» — оставить только H2 «Рассчитать стоимость».
- Уменьшить высоту триггера селекта с `h-14` до `h-12`.

### 6. Анимации между шагами
Оставить `framer-motion` AnimatePresence, но переходы сделать мягче: `x: 12` вместо `x: 20`, `duration: 0.2`.
